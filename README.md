# 시설 진단하기 (Facility Diagnosis App)

시민 및 전문가가 시설물의 상태를 효율적으로 진단하고 평가할 수 있도록 돕는 React 기반 웹 애플리케이션입니다. 엑셀 파일을 통해 유연하게 데이터를 관리하며, 직관적인 단계별 UI를 제공합니다.

## 🚀 주요 기능

### 1. 동적 데이터 로딩
- `xlsx` 라이브러리를 사용하여 엑셀 파일(`체크리스트(시민, 전문가)_251205_싱크앤두랩_수정.xlsx`)에서 직접 데이터를 읽어옵니다.
- **대분류 -> 중분류 -> 질문**으로 이어지는 데이터 구조를 자동으로 파싱하여 UI에 반영합니다.

### 2. 시설 선택 (Facility Selection)
- 대분류 선택 시 해당하는 중분류 리스트가 이미지 카드 형태로 표시됩니다.
- 엑셀 데이터의 카테고리 명을 분석하여 알맞은 이미지 아이콘을 자동으로 매핑합니다.

### 3. 만족도 평가 (CheckList)
- 선택한 중분류에 해당하는 체크리스트 문항이 동적으로 생성됩니다.
- **유효성 검사**: 모든 문항에 대해 만족도(1~5점)를 평가해야만 다음 단계로 이동할 수 있습니다.
- 커스텀 디자인된 라디오 버튼과 슬라이더 UI를 제공합니다.

### 4. 리뷰 작성 (Review)
- 시설에 대한 추가 의견을 작성하는 페이지입니다.
- **글자 수 제어**:
  - 최소 **20자 이상** 입력해야 제출 가능 (버튼 활성화).
  - 최대 **1,600자**까지 입력 가능하며, 실시간으로 글자 수가 카운트됩니다.

### 5. 진단 완료 (CheckDone)
- 모든 과정이 완료되었음을 알리는 결과 페이지입니다.
- 직관적인 완료 아이콘과 메시지를 제공하며, 홈으로 복귀할 수 있는 버튼을 포함합니다.

### 6. 모바일 최적화 (Mobile Responsive)
- **홈 화면 레이아웃**:
  - 모바일 환경에서 [제목/본문 → 지도 → 버튼 그리드] 순서의 세로형 레이아웃을 제공합니다.
  - 배경 이미지가 콘텐츠 길이에 따라 유동적으로 확장되며, 버튼 그리드가 자연스럽게 오버랩됩니다.
- **테블릿/데스크탑**: 1024px 이상 해상도에서는 가로형 레이아웃으로 자동 전환됩니다.

### 7. 위치 선택 디자인 (Location Select)
- **부산 지도 배경**: 진단 시작 시 위치를 선택하는 화면에 부산 지도 배경 이미지를 적용하여 시각적 몰입도를 높였습니다.
- 직관적인 핀 인터페이스를 제공합니다.

## 🛠 기술 스택
- **Core**: React 18+, Vite
- **Language**: JavaScript (ES Modules)
- **Styling**: Vanilla CSS (CSS Modules 패턴 지향)
- **Data Parsing**: SheetJS (`xlsx`)

## 📦 설치 및 실행

### 필수 요건
- Node.js (v14 이상 권장)
- npm

### 1. 프로젝트 클론 및 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
로컬 환경에서 개발 서버를 실행합니다.
```bash
npm run dev
```

### 3. 프로덕션 빌드
배포를 위한 정적 파일을 생성합니다.
```bash
npm run build
```
